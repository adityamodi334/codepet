# CodePet 🐾

## About

CodePet is a my first ever aproper project I built to make coding feel more fun and less like a chore.
The idea is simple — instead of just tracking your study time, you get a **virtual pet that grows as you code**.

So the more consistent you are, the more your pet evolves. Kind of like a POKEMON, but for productivity.

## ⚙️ Features

* 🥚 Pet evolution system based on XP
* ⏱️ Pomodoro timer for focused sessions
* 🔥 Streak & combo system
* 📊 Activity tracking (sessions, hours, etc.)
* 🏆 Leaderboard (to compare with others)

## 🛠️ Tech Stack used

* **Backend:** FastAPI (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Storage:** (handled via backend API + localStorage for some data)

## 💡 Why I made this

I noticed that tracking study time gets boring really fast.
So I wanted to build something that gives a **visual reward** for consistency. And also gave us the feel of enjoyment while studying.

Instead of “I studied 2 hours”, it feels like
👉 “my pet just leveled up because I showed up today”

## ▶️ How to run

1. Start the FastAPI server:

   ```bash
   uvicorn main:app --reload
   ```

2. Open the frontend (`index.html`) in your browser

3. Enter a username and pet name

4. Start logging sessions and watch your pet grow 🚀
